# JSON → BSON Converter

JSON 파일을 BSON으로 변환하고 메타데이터를 생성합니다.

## 사용 방법

1. **`input`** 폴더에 `.json` 파일을 넣으세요
2. **`convert.bat`** 더블클릭
3. **`output/날짜시간/`** 폴더에서 결과 확인:
   - `파일명.bson` - 변환된 BSON 파일
   - `파일명_metadata.json` - 상세 메타데이터

## 기능

- ✅ 대용량 파일 지원 (16MB 스트리밍)
- ✅ 단일/다중 문서 자동 구분
- ✅ 메타데이터 자동 생성
- ✅ 데이터 타입 자동 분석
- ✅ 압축률 비교

## 폴더 구조

```
json-to-bson/
 ├── input/          ← JSON 파일 넣는 곳
 ├── output/         ← BSON + 메타데이터 나오는 곳
 ├── convert.js      ← 변환 스크립트
 └── convert.bat     ← 실행 파일
```

## 메타데이터 정보

생성되는 메타데이터에는 다음 정보가 포함됩니다:
- 원본 파일 정보
- 변환 일시
- 문서 개수 및 구조
- 각 문서의 필드 및 타입
- 압축률 (JSON vs BSON)
