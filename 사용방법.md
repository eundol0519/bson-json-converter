# 📖 사용 방법 (간단 가이드)

---

## 🔵 BSON → JSON 변환

### 1️⃣ BSON 파일 준비
`bson-to-json/input/` 폴더에 변환하고 싶은 `.bson` 파일을 넣으세요.

### 2️⃣ 변환 실행
`bson-to-json/convert.bat` 파일을 더블클릭하세요.

### 3️⃣ 결과 확인
`bson-to-json/output/날짜시간/` 폴더에 생성된 `.json` 파일을 확인하세요.

---

## 🟢 JSON → BSON 변환

### 1️⃣ JSON 파일 준비
`json-to-bson/input/` 폴더에 변환하고 싶은 `.json` 파일을 넣으세요.

### 2️⃣ 변환 실행
`json-to-bson/convert.bat` 파일을 더블클릭하세요.

### 3️⃣ 결과 확인
`json-to-bson/output/날짜시간/` 폴더에서 결과를 확인하세요:
- `파일명.bson` - 변환된 BSON 파일
- `파일명_metadata.json` - 상세 정보

---

## 💡 참고사항

- **첫 실행 시**: Node.js 패키지가 자동으로 설치됩니다 (조금 기다려주세요)
- **여러 파일**: 한 번에 여러 파일을 넣어도 모두 변환됩니다
- **결과 위치**: 변환할 때마다 새로운 날짜시간 폴더가 생성됩니다
